## Стратегические замечания

1. **КвантоКод — флагман по SEO-приоритету.** Несмотря на скромный объём ядра (~1 945 точных показов/мес), это:
   - Узкая ниша с низкой конкуренцией (особенно по «квантовая нумерология», «код души рассчитать»).
   - Высокая конверсия в действие (целевая аудитория ищет именно расчёт).
   - Отлаженная воронка с фискальным чеком и средним чеком от 490 ₽ до 2 490 ₽.

2. **Сравнение продуктов по SEO-потенциалу:**
   - КвантоКод: ~1 945 точных показов/мес (узко, флагман).
   - Mystic Oracle: ~115 000 (массово, но воронка не отлажена, приоритет англ.).
   - ЭЛИОР: ~4 470 (среднее, инфо-трафик с низкой конверсией).

3. **Реалистичный прогноз:** при выходе в топ-10 по 5-7 ключам ожидаемый трафик — 50-150 уникальных посетителей/мес из поиска через 4-8 месяцев. Это не «трафик», это **квалифицированные лиды**: каждый второй-третий перейдёт в бот.

4. **Расширение ядра** — через 1-2 месяца, когда из Метрики и Вебмастера придут реальные поисковые запросы, по которым люди заходят. Часть из них окажется неожиданной — это будет ценная информация для следующей итерации.
