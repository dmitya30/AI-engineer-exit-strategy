
# Фронтенд — текущее состояние

**Версия:** 1.1
**Дата:** 01.06.2026
**Статус:** разработка завершена в первой итерации, сайт в продакшене

## 1. Архитектура стилей

Единый источник стилей — Code Injection → Site Header в Ghost. Локальные `<style>` на страницах удалены, кроме одного: html-карточка главной страницы (`/`) содержит уникальную лендинговую разметку с классами `.td-section-out`, `.td-hero`, `.td-features` и т.д.

Все правила для страниц и постов написаны через зеркалирование на нативную структуру Ghost: `.gh-content > h2`, `.gh-content > p`, `.gh-content table` и т.д. Markdown-посты, html-card-страницы и нативные Koenig-блоки рендерятся одинаково.

Устаревшая обёртка `<article class="td-page">` удалена со всех страниц и постов. Свойства перенесены на уровень `.gh-content`. Из утилит сохранены независимые классы: `.td-page-cta`, `.td-byline`, `.td-num`, `.td-table`, `.td-section-*`, `.td-btn*`.

## 2. Дизайн-система

Дизайн-токены в `:root` как CSS-переменные. Палитра: фон `#FAF8F4`, вторичный `#EFE9DD`, текст графит `#1F2937`, акцент бирюза `#0F4C5C`, тёплый акцент терракота `#C97B4F`. Ширины контейнеров: читательская 720px, узкая 880px, широкая 1100px.

Скрыты системные элементы темы Source: переключатель тёмной темы, кнопка Subscribe, форма подписки, футер темы, мета-информация автора и даты в статьях. Тёмная тема принудительно отключена.

## 3. Типографика

Шрифты Lora 500/600 (заголовки) и Inter 400/500/600 (текст). Подключены через Google Fonts с preconnect. Базовый размер 18px / 1.65. H1-H4 через `clamp()` для плавного масштабирования.

## 4. Шапка и навигация

Логотип Lora 600, размер через `clamp()`. Меню Inter 500, размер 18px. Цвета: графит → бирюза (активный) → терракота (hover). Решена проблема `rem`-приоритета темы Source через специфичные селекторы `.gh-navigation` + `!important`.

Логотип реализован как `<img src="/content/images/2026/05/logo-full-1.svg">` внутри `.gh-navigation-logo.is-title` (тип шапки is-middle-logo). Alt берётся из Site title — «Квантовая нумерология — расчёты по дате рождения | Тихий Дом». Ограничение `max-height: 44px` задано в Code Injection. Оригинал файла — `logo-full.svg`, в Ghost загружен как `logo-full-1.svg`. Favicon — `favicon.svg`, отдаётся Ghost в формате PNG 256×256 через путь `/content/images/size/w256h256/format/png/2026/05/favicon.svg`. Размер шрифта меню — 18px.

## 5. Контентные шаблоны (страницы и посты)

Единая ширина 680px (контейнер 720px минус padding 20px с каждой стороны) для title, excerpt, контента, featured image. На мобильных адаптивная через `box-sizing: border-box`, без отдельных `@media`.

Title и excerpt центрированы. Grid `.gh-canvas` от темы Source отключён через `display: block` для всех шаблонов кроме главной. Аналогично для `.gh-article-header`.

Стилизованы все Markdown-элементы: p, h2-h4, blockquote (терракотовая левая полоса), ul/ol, strong, em, inline-code, pre, hr (короткий разделитель 100px по центру), таблицы. Подпись автора после `<hr>` — мелкий серый курсив.

## 6. Таблицы

Решена проблема с обёрткой `<div class="gh-table">` от темы Source. По умолчанию даёт `overflow-x: auto` + `white-space: nowrap` — лишний скролл. Нейтрализовано: ячейки переносятся, последняя колонка фиксируется (`nowrap`, `width: 1%`, right). Шрифт заголовков 16px / 500, ячеек 17px / 400; на мобиле 15px / 15px.

## 7. Кнопки

Класс `.td-btn` с модификаторами размера (`--sm / --lg`) и стиля (`--primary` бирюза, `--warm` терракота). CTA-обёртка `.td-page-cta`. На hover — перекраска + translateY + усиление тени.

## 8. Главная страница

Уникальная вёрстка через `.td-section-out` — секции на всю ширину viewport с центрированными контейнерами `.td-section-inner` (узкий 880px / широкий 1100px). Решена проблема «съезда вправо» на мобиле — `margin-left: calc(50% - 50vw)` вместо `width: 100vw`.

Стандартный article-header Ghost (дублирующий H1 «Главная») скрыт через `position: absolute; clip: rect(0,0,0,0)` — остаётся в DOM для SEO/OG, не виден.

Обнулены `margin-top` у детей `.gh-content` на главной — иначе тема через `.gh-content > * + * { margin-top: 28px }` давала фантомные разрывы между секциями.

## 9. Адаптивность

Единая мобильная контрольная точка `@media (max-width: 720px)`. На мобиле: padding 16/40 вертикальный, шрифты уменьшаются (p 17px, h2 26px, h3 21px), таблицы 15px. Главная имеет собственный набор правок в html-карточке.
