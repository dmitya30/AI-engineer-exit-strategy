## Состояние: START (первый запуск / перезапуск)

### Обычный пользователь
- **Триггер:** команда `/start`
- **Текст сообщения:**
```
🔮 Цифры знают о тебе больше, чем ты думаешь.

КвантоКод рассчитает твои персональные коды по дате рождения и ФИО, покажет, какие энергии в тебе преобладают и что именно тебе лучше всего монетизировать.

Что умеет бот:
• Расчёт кодов души — бесплатно
• Полный расчёт: душа + тело + личность с AI-интерпретацией
• Проверка совместимости: название, телефон, проект, человек
• Подбор идеального названия с кодом денег

Для экспертов: режим безлимитных расчётов + PDF

👇 Введи дату рождения (ДД.ММ.ГГГГ):
```
- **Кнопки:** нет
- **Переход:** → состояние AWAITING_BIRTHDATE

### Эксперт / VIP
- **Триггер:** команда `/start` (при `role='vip'` или `expert_until > NOW()`)
- **Текст сообщения:**
```
🔮 Режим эксперта

Введи данные клиента одним сообщением:
```

## Состояние: AWAITING_BIRTHDATE

- **Триггер:** пользователь вводит текст
- **Валидация:** формат ДД.ММ.ГГГГ, корректная дата, год 1900–текущий
- **При ошибке:** текст с описанием ошибки, остаёмся в AWAITING_BIRTHDATE
- **При успехе:** бот рассчитывает Код Души, запрашивает тизер из БД, считает общий счётчик расшифрованных кодов
- **Текст ответа (пример для кода 23):**
```
✨ Твой Код Души: 2, 6, 1, 9, 5 → 23
Успех через совесть и справедливость

«Код 23 — код справедливого успеха...» (тизер из БД)

🔮 Расшифровано кодов: 1 247

💡 Это лишь верхушка айсберга. Полная расшифровка всех трёх кодов (душа + тело + личность) с детальной AI‑интерпретацией покажет полную картину твоих энергий.
```
- **Переход:** → проверка подписки на канал

- **Триггер:** автоматически после расчёта Кода Души
- **Проверка:** бот вызывает Telegram API `getChatMember` для канала @quantocode

## Состояние: оплата (любой тариф)

- **Триггер:** нажатие кнопки `buy_basic`, `buy_check`, `buy_generate`, `buy_expert_day`, `buy_expert_week`, `buy_expert_month`
- **Действие:** бот отправляет Telegram Invoice (нативная оплата Stars)
- **Тарифы и цены:**
  - basic — 325 ⭐ «Все мои коды»
  - check — 532 ⭐ «Проверка совместимости»
  - generate — 1529 ⭐ «Подбор названия»
  - expert_day — 5725 ⭐ «Эксперт: День»
  - expert_week — 22975 ⭐ «Эксперт: Неделя»
  - expert_month — 59459 ⭐ «Эксперт: Месяц»
- **При успешной оплате:** → переход зависит от тарифа (см. ниже)
- **При отмене:** Invoice закрывается, кнопки предыдущего сообщения остаются

## Состояние: генерация ТОП-10 названий

- **Триггер:** нажатие кнопки `goal_*`
- **Действие:** бот показывает «typing...», LLM генерирует ~30 000 вариантов, скоринг отбирает ТОП-10 по формуле (0.5×деньги + 0.3×тело + 0.2×канал), AI-интерпретация
- **Текст ответа:** ТОП-10 названий с кодами и оценками + AI-интерпретация
- **Кнопки:**
  - [📄 Скачать PDF] → callback `get_pdf`

- **Переход:** → состояние GEN_DONE

## Нештатные ситуации

- **Текст вместо кнопки в EXPERT_READY:** бот отвечает «☝️ Используй кнопки меню:» и показывает хаб с кнопками заново
- **Стикер / фото / голосовое:** бот отвечает «🔮 Напиши /start для возврата в меню эксперта.» (для эксперта) или «🔮 Напиши /start чтобы начать.» (для обычного)
- **Покупка check/generate без body_code:** бот блокирует и предлагает сначала купить полный расчёт
- **Истечение подписки эксперта:** при следующем `/start` показывается обычное меню
- **VIP (role='vip'):** всегда в режиме эксперта, подписка не истекает (expert_until = +100 лет), устанавливается вручную админом
