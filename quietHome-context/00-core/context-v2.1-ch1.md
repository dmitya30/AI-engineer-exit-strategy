## 1. Бренд и позиционирование

**Зонтик:** «Тихий Дом» — экосистема эзотерических продуктов с нумерологией как ядром. Название и идея зонтика — авторство Александра Романова (Саныча).

**Флагман:** КвантоКод — бот квантовой нумерологии (`https://t.me/quantocode_bot`, канал `https://t.me/quantocode`).

**Слоган:** «Числа всерьёз. Расчёты, которым доверяешь.»

**Сайт:** https://numerologpro.ru (в продакшене с 11.05.2026).

**Внутренний камертон (НЕ публичный):** «Тихий Дом матушки-нумеролога с уютными уголками по интересам». Используется как проверка тона. Наружу не выводим — слово «матушка» сужает ЦА.

**Внутренний концептуальный каркас (НЕ публичный):** «Игрок и персонаж» — метафора Алекса Крола. Душа = игрок (приходит за опытом), Тело = персонаж (биология, гормоны, выживание), Личность = интерфейс между ними. Используется как логическая основа описаний трёх кодов в КвантоКоде. Запланирована статья «Игрок и персонаж: квантовая нумерология глазами наблюдателя» на месяц 2-3 после запуска — авторская визитка Димы.

**Тон коммуникации:** «Разговор подруги-эксперта на кухне». Тёплый, дружелюбный, без «я-учу-ты-слушай», с бытовыми метафорами и прямой речью героев. Не пятиклассник, не пафос.

**Три голоса проекта** (распределение ролей в текстах):
- **Дима — инженер-наблюдатель.** Голос для скептиков и физиков. Точные формулировки, бытовые метафоры (без инженерного жаргона), позиция наблюдателя без догм.
- **Наташа — практик.** Голос для тех, кто доверяет и применяет. Мягкие методические тексты, рабочие метафоры (дом → дизайн-проект → материалы → как распорядился), ведический пласт дозированно.
- **Саныч — ироничный рассказчик.** Голос для тех, кто ценит культурный слой. Цитаты, отсылки (Пелевин, Шолохов), самоирония, формат «Я — домовой. Шутка».

Каждая публичная статья и пост подписываются автором (имя + роль). Редакционные тексты подписываются «Тихий Дом».

**Принцип отстройки — через позитивное утверждение, не через критику:**
> «Мы про точные расчёты, мы про математику, мы не предсказываем».

Не строим позиционирование через «мы не такие, как [плохие нумерологи]». Это решение авторов проекта. Наташа из роли критика рынка снята — это не её природа.

**Двойная ЦА** — одно сообщение, два слоя:
- Основной слой — для физиков (женщины 30-45, интересующиеся нумерологией для себя).
- PS-крючок — для практикующих нумерологов (B2B, режим Эксперт).

**Метод написания текстов** (7 находок):
1. Имя героя (Аня, Маша, Катя, Лена) — эмпатия.
2. Структура «боль → разъяснение → решение».
3. Цитата героя в кавычках от первого лица — эмоциональный пик.
4. Бытовая метафора вместо термина («рюкзак, который ты не складывала» вместо «энергетический блок»).
5. Рваный ритм: длинные описания + короткие ударные фразы.
6. Технические факты (коды, цены) внутри сюжета.
7. Финальная реплика героя или короткая фраза, затем CTA.

## 2. Стоп-список и языковые правила

> **Область применения:** правила этого раздела распространяются ТОЛЬКО на публичные тексты проекта - посты в каналах, сторис, статьи блога, страницы сайта (включая /about/, тарифы, FAQ), тексты бота и любые материалы, которые видит конечный пользователь.
>
> **На служебную документацию (state-current, планы, профили команды, оркестратор, аналитика, контекстные артефакты, README, changelog и т.п.) эти правила НЕ распространяются.** Служебка - техническая документация, в ней допустимы эмодзи, длинные тире, ёлочки, любая структура и любой уровень формальности. Главное - читаемость и точность.

**Слова под запретом:** AI, нейросеть, Gemini, JavaScript, технология (как самоназвание), магия, гадание, эзотерика (как самоназвание; упоминание ниши — ОК).

**Инженерный жаргон Димы — под запретом в текстах Тихого Дома:** архитектура, алгоритм, система, интеграция, инженерный, кибернетика, миллисекунды, методика, профилирование, оттрейсить, английские термины (кроме названий продуктов). Дима пишет «Тихий Дом» с кухни, не из серверной.

**Ведический пласт Наташи** (Навь/Явь/Правь, васту, ведическая нумерология/астрология) — допустим, но дозированно: только когда работает на объяснение метафоры, не как декор.

**Формулировки под запретом:** «Не гадание. Не магия.» и любые негативные конструкции через отрицание; технический жаргон без перевода; рекламно-восторженный тон.

**Что говорим вместо «AI»:** «инструмент», «бот считает по формулам квантовой нумерологии», «расчёты построены на оригинальной методике», «как калькулятор инженера».

**Типографика:**
- Длинные тире (—) — запрещены, использовать короткий дефис (-).
- Кавычки-ёлочки («») — запрещены, использовать прямые ("").
- Эмодзи — максимум 1-2 на абзац.

**Адресация — гибридная:**
- Базовое обращение — **на «вы»** (вежливое, не сужает аудиторию).
- В эмоциональных точках (заголовки-слоганы, цитаты от первого лица, прямая речь героев, разговор с конкретным «одним» читателем) — допустимо **«ты»**.
- Принцип: «вы» — ко всем, «ты» — к одному. Не смешивать в пределах одного блока.
- Слоган «Числа всерьёз. Расчёты, которым доверяешь» — остаётся в «ты», работает как эмоциональный крючок.

**Правило ссылок:** все Telegram-ссылки в публичных текстах — только через `https://t.me/...`. Никаких `@username` (они работают только внутри Telegram, на сайте/Дзене/MAX превращаются в нерабочий текст).

## 5. Сайт numerologpro.ru — архитектура и реализация

**Статус:** в продакшене с 11.05.2026. Готов к продвижению.

**Инфраструктура (важно — две разные площадки):**
- **FastVPS** (Москва, ~190 ₽/мес, 1 vCPU Intel Xeon, 2 GB RAM, 10 GB SSD): хостинг сайта (Ghost CMS), Docker, SSL Let's Encrypt. Сайт работает как «информационная оболочка» без принудительного сбора данных — все формы отключены через Code Injection.
- **Beget** (KZ): инфраструктура ботов (n8n). Юр-факт для ЮKassa: ЮKassa зарегистрирована на ТГ-бота, не на сайт. Политика конфиденциальности корректно ссылается именно на Beget.

**Разработка:**
- **Backend** — закрыт в отдельном диалоге, поддержка по запросу.
- **Frontend** — закрыт в отдельном диалоге, отчёт получен. Дальнейшие правки — итеративно в том же диалоге.

**Стратегическая логика:** трафик собираем на сайт, не в TG-канал (риски блокировок). Все каналы (Дзен, MAX, посевы, VK) ведут на сайт, оттуда в `https://t.me/quantocode_bot`.

**Архитектурное решение (вариант В + 2С):**
- Сейчас `/` = лендинг КвантоКода под зонтиком «Тихий Дом» (карточки «Уголки Тихого Дома»).
- При появлении второго продукта `/` → зонтичная главная, КвантоКод → `/quantocode/` через 301.
- Через 2-3 месяца финальное решение по доменной архитектуре.

**URL-структура:**
```

/blog/               — список статей (опубликован)

/about/              — о Тихом Доме (опубликован, ~3500 знаков)
```

**Юр-документы (вариант 2):** дубль на сайте с canonical на Telegraph. Telegraph остаётся первичным (юр-факт для ЮKassa и налоговой). На каждой странице плашка: «Это копия для удобства. Приоритет — оригинал на Telegraph».

**CMS:** Ghost 5.130, тема Source v1.5 (адаптирована).

**Дизайн-система:** реализована в Code Injection как единый источник правды (~35 KB CSS). Токены палитры, ширины колонок (`--td-w-wide` 1100 px, `--td-w-narrow` 880 px, `--td-w-reading` 720 px), радиусы, тени, переходы. Перекрытие переменных темы Source для принудительного light-режима. Подробности — в Фирменном стиле и визуальном гайде v1.0.

**Структура главной (9 блоков, опубликована):**

**Реализация Hero:** не сгенерированная картинка, а связка **WebP-фон** (paper texture + акварельная сетка + тень листьев в углу, 16:9, 2560×1440 px) **+ HTML-числа 23/30/17** с разной opacity (0.18-0.20) и последовательной CSS fade-in анимацией (`@keyframes td-num-in`, задержки 200/400/600 мс). `prefers-reduced-motion` учтён.

**Featured-картинки статей:** единый авторский стиль (акварельные органические формы + инженерная сетка + числа в фирменных цветах), не сток.
