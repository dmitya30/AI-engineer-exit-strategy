## 2. Отвергнуто окончательно (не возвращаться)

- **Классический арбитраж трафика на гемблинг (модель Гришина).** Серая ниша, не близка по ценностям.
- **Создание своего софта нейрокомментинга как продукта для продажи.** Рынок плотный (GramGPT, Botseller, Easy AI, Neurocom.store, Socrupor, TgNinja, Virality, Neyrokommenting.ru), нет навыка маркетинга B2B-SaaS. Принято: софт = персональный инструмент, не товар.
- **Классический офисный найм.** По жизненным предпочтениям и реалиям рынка для 50+. Удалённый part-time/контракт — возможно, если припрёт.
- **Англоязычный рынок как быстрый канал.** Нет устного английского, нет навыка работы с иностранным заказчиком. Откладывается на неопределённое будущее.
- **YouTube-канал «с лицом» про себя.** Не готов сниматься на камеру. Faceless с собственным голосом — допустимо, но сейчас на парковке (см. 1.1).
- **Глубокий разбор канала Гришина по таймкодам.** Неактуально.
- **Полноценный B2B-лидген с продажами для IT-компании.** Только лид-скаут на ретейнере (см. `20-it-leadgen-context.md`). Продажи — на стороне заказчика.
- **WhatsApp/MAX полноценные интеграции для проекта Елены.** Передача контактов текстом, не интеграция.
- **Хранилище контекста в GitHub Gist.** Crawler LLM плохо рендерит кириллицу в Gist. Используется публичный репозиторий.
- **Полная замена документов при правках.** Только точечные патчи / diff / `cat >` для перезаписи. Не выдавать полные тексты ради «удобства».
- **Жёсткий тайм-менеджмент «делить день пополам».** У Дмитрия не работает, проверено.
- **Использование `sed` для редактирования.** Плохо работает с regex. Find-and-replace, find-and-delete, add-before/after, diff.
- **Принятие предложений под давлением "внутреннего прокурора".** Любая сделка, которая решает не задачу "деньги на иждивение в неделях", а задачу "успокоить чувство вины прямо сейчас" - отвергается. Концепт, защитная конструкция и три проверочных вопроса - в `10-self-saboteur-debrief.md`, раздел "Дополнение от 30.05.2026: внутренний прокурор". Первый зафиксированный кейс срабатывания - история с MVMTech (см. `20-it-leadgen-context.md`).

## 3. Открытые вопросы (требуют решения, но не сейчас)

- **Юридическое оформление работы** (ИП / самозанятый / ООО). При чеках 100к+ становится релевантным. Решение — после первого крупного заказа.
- **Выбор приоритетной отрасли** для будущего outbound (если ветка разморозится).
- **Контент-маркетинг и автоматизация VC.ru:** пайплайн с AI-агентом приостановлен из-за отсутствия API-интеграции. Решение — обходим (полуавтоматическая публикация) или нет.
- **Подушка/cashflow в ближайшие 30 дней,** пока Квантокод не выстрелил. Источники: текущие 30к + потенциальные 50–70к от Елены = 80–100к на месяц. Этого хватает, но запаса нет.
